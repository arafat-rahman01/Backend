# Mini Post Management System

Description
A RESTful CRUD application built using Node.js, Express, and EJS to understand backend fundamentals and server-side rendering.

Technologies
- Node.js
- Express.js
- EJS
- UUID
- Method Override

Features
✔ Create posts  
✔ View all posts  
✔ View single post  
✔ Update posts  
✔ Delete posts  
✔ RESTful routing  
✔ Dynamic rendering  

What I Learned
- Routing in Express
- Middleware handling
- HTTP methods (GET, POST, PATCH, DELETE)
- Template rendering with EJS
